# Darkrise Astro (Netzwerk)

Commands:
- npm install
- npm run dev  # http://localhost:4321
- npm run build && npm run preview

Tech:
- Astro + Tailwind (dark-first), Typography plugin
- Simple scroll reveals + view transitions
- Pages: /, /services, /about, /contact

Edit theme: tailwind.config.mjs (accent color under theme.extend.colors.accent)
